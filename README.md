# UC2: Toss to Decide First Player and Symbol

## Goal
Randomly decide which player starts and assign symbols (X or O).

## Concepts Used
- Random Number Generation
- Conditional Logic
- Game State Variables

## How it Works
- System performs a toss
- One player is selected randomly
- Symbols are assigned
- Current player is stored

## Output
Displays:
- Toss winner
- Symbols
- First player
